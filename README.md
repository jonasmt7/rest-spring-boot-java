# REST API com Spring Boot 3 e Java 21

Projeto desenvolvido durante o curso **Spring Boot 2025: Do Zero à Amazon AWS e Google Cloud com Java, Docker e Kubernetes**, com foco na construção de APIs RESTful utilizando as melhores práticas do ecossistema Spring.

## 🚀 Tecnologias Utilizadas

* Java 21
* Spring Boot 3
* Spring Data JPA
* Spring Security
* JWT Authentication
* Hibernate
* Flyway
* MySQL
* Swagger / OpenAPI
* Docker e Docker Compose
* JUnit 5
* Mockito
* REST Assured
* TestContainers
* GitHub Actions
* AWS e Google Cloud Platform (GCP)

## 📚 Conceitos Aplicados

* Arquitetura em camadas
* API RESTful
* Injeção de Dependências
* DTO Pattern
* Builder Pattern
* Factory Pattern
* Strategy Pattern
* Tratamento Global de Exceções
* Paginação e Ordenação
* HATEOAS
* Content Negotiation
* Versionamento de APIs
* Upload e Download de Arquivos

## 🔐 Segurança

A aplicação utiliza:

* Spring Security
* Autenticação via JWT
* Controle de acesso baseado em permissões
* Configuração de CORS

## 🧪 Testes

Foram implementados testes para garantir a qualidade e confiabilidade da aplicação:

* Testes Unitários com JUnit 5 e Mockito
* Testes de Integração
* Testes de API com REST Assured
* Banco de dados isolado com TestContainers

## 🐳 Executando com Docker

```bash
docker-compose up -d
```

## ▶️ Executando o Projeto

### Clonar o repositório

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

### Entrar na pasta do projeto

```bash
cd seu-repositorio
```

### Executar a aplicação

```bash
./mvnw spring-boot:run
```

Ou:

```bash
mvn spring-boot:run
```

Durante o desenvolvimento foram explorados conceitos de:

Este projeto tem como objetivo consolidar conhecimentos em desenvolvimento backend com Spring Boot, aplicando práticas modernas de mercado para construção, testes, segurança, conteinerização e implantação de APIs escaláveis.
